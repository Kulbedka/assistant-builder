# Assistant Builder

Fullstack platform for creating and managing AI assistants.

## Live Demo

https://assistant-builder-beige.vercel.app

## Tech Stack

- Next.js
- TypeScript
- Express.js
- Prisma
- PostgreSQL
- Supabase
- REST API
- Vercel

## Features

- View available assistants
- Create new assistants
- Open chat with an assistant
- Send messages
- Store assistants in PostgreSQL
- Store chat messages in PostgreSQL
- Connect frontend with external Express API

## Architecture

Frontend: Next.js on Vercel  
Backend: Express.js API on VPS  
Database: PostgreSQL via Supabase  
ORM: Prisma
